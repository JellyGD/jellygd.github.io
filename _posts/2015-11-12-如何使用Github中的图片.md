* 将图片放在仓库里面，在文件里链接它，最后 push 到 github 上。
* github 图片的链接如下：https://raw.githubusercontent.com/`yourname`/`your-repository`/`master`/`images-folder`/`xx`.jpg


yourname -- 你的账号


your-repository --你的项目名称


master --分支


images-folder -- 图片的文件夹，如果直接放在工程项目下，可以省略


xx --图片名称
